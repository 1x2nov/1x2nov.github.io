Project 3

프로젝트 명

프로젝트 개요

VMWARE ESXi를 활용한 고가용성 서비스 구축

개인/팀 구분(역할)

팀 프로젝트 (PM)

구현 기능

1. 인프라 및 서비스의 이중화를 통한 고가용성 확보
2. VMWARE ESXi를 사용한 가상서버 운용
3. Zabbix를 통한 모니터링 환경 구축
4. NAT를 통한 내/외부망간 통신 구성
5. DR구성으로 운영 안정성 확보
6. NFS서버를 통한 운영 ↔ DR 간 데이터 동기화 구성

Review

1. 초기 목표로 했던 대부분의 기술을 구현 했으나 개발 관련된 부분의 숙련도 부족으로 구현에 실패한 부분이 아쉬움.
2. Oracle CDC를 사용했던 경험을 토대로 운영↔DR 간 데이터 복제를 구현하고자 했으나, 배치로 원하는 수준의 구현을 못한 부분이 아쉬움.
3. 네트워크, 서버 인프라 구성을 대부분 맡아 진행하며 처음 혹은 부족했던 영역을 구축하는 과정에서 기술적 숙련도 향상이 있어 좋았음.

Project 4

프로젝트 명

프로젝트 개요

AWS Cloud 와 On-premise(k8s)를 기반으로 한 고가용성 Hybrid Cloud 구축

개인/팀 구분(역할)

팀 프로젝트 (PM)

구현 기능

1. AWS Cloud 내 EC2를 활용한 웹서버 구축
2. VPC peering을 사용한 각 Region별 EC2간 통신 확보
3. ALB 구성으로 EC2 내 서비스의 HA 확보
4. AWS Client VPN을 통한 AWS - On-premise 간 네트워크 구성
5. K8S multi-master 구성 및 동적 PV할당으로 HA 확보

Review

1. Free Tier 로 제한된 환경에서 구성함에 따라 안정성이 생각보다 저하됨.
2. 스케줄 및 리스크 관리에 대한 디테일 부족
3. AWS Cloud에서 제공하는 솔루션에 대한 활용이 매끄럽지 못한 점이 아쉬움.
